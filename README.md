搬运整合源：  https://github.com/TinhoXu/BookSource/raw/main/sourceModelList.xbs  

PS：
1. 书源来源于网络，如有侵权请联系删除
2. 书源仅供个人学习与交流使用，严禁用于商业以及不良用途
3. 您必须在下载后的24小时内从计算机或手机中删除书源内容
